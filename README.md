# CreditNow - Instant Loan Application

This is a web application for "CreditNow," a fictional loan provider. It serves as a comprehensive landing page to showcase various loan products, highlight special offers, display customer testimonials, and allow potential customers to apply for loans quickly.

## ‚ú® Features

- **Responsive Design:** Fully responsive layout that works on desktop, tablet, and mobile devices.
- **Product Showcase:** A detailed section displaying all available loan products, such as Two-Wheeler Loans, Used Car Loans, and Personal Loans.
- **Special Offers Carousel:** An interactive carousel to highlight current promotions and offers.
- **Quick Apply Forms:** Multiple forms for users to quickly apply for loans or check for pre-approved offers.
- **Customer Testimonials:** A section to build trust by showcasing positive feedback from customers.
- **Company Journey:** A visual representation of the company's milestones and growth.
- **Chatbot FAB:** A floating action button for a virtual assistant (TIA) to provide support.
- **Modern UI:** Built with modern design principles using ShadCN UI and Tailwind CSS.

## üöÄ Getting Started

Follow these instructions to set up and run the project locally.

### Prerequisites

- [Node.js](https://nodejs.org/en/) (version 20 or later recommended)
- [npm](https://www.npmjs.com/) (usually comes with Node.js)

### 1. Installation

Clone the repository and install the dependencies:

```bash
npm install
```

### 2. Running the Development Server

To start the local development server, run the following command:

```bash
npm run dev
```

The application will be available at [http://localhost:9002](http://localhost:9002).

### 3. Building for Production

To create a production-ready build of the application, run:

```bash
npm run build
```

This will generate an optimized version of the app in the `.next` directory.

### 4. Running in Production Mode

To start the application in production mode after building, use:

```bash
npm start
```
