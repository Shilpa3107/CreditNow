# **App Name**: CreditNow

## Core Features:

- Loan Type Selection: Allow users to select the type of loan they are interested in from a dropdown.
- Product Showcase: Display various loan products with images and brief descriptions, such as Two Wheeler Loans, Used Car Loans, etc.
- New/Existing Customer Toggle: Enable users to identify themselves as new or existing customers.
- Quick Apply: Allow a 'quick apply' of loans.
- TIA Chatbot: Integrate a chatbot named TIA for instant customer support and loan information.

## Style Guidelines:

- Primary color: Forest green (#388E3C) to convey trust, growth, and financial well-being.
- Background color: Light sage (#F1F8E9), a very lightly saturated tint of the primary green, to create a calming and trustworthy backdrop.
- Accent color: Yellow-Green (#9ACD32), to highlight calls to action and important information.
- Body and headline font: 'PT Sans', a humanist sans-serif font combining a modern look with warmth or personality; use for headlines and body text.
- Use clean and modern icons to represent different loan products and features.
- Maintain a clean and structured layout with clear sections for loan selection, product display, and customer support.
- Subtle animations on hover and click events to enhance user engagement and provide visual feedback.
